# network-traffic-pipeline

## Pokretanje projekta

Projekat se pokrece pomocu sbt buildera.
Potrebno je instalirate Javu, Scalu, sbt i dodati sve u PATH.
Iz korenog repozitorijuma projekta komandom

```` terminaloutput
sbt
````

Pokrece se sbt Shell iz koga posle mozemo vrsiti pokretanje projekta, kompajliranje, rebuild i testiranje i to komandama:

```` terminaloutput
run
````

```` terminaloutput
compile
````

```` terminaloutput
rebuild
````

```` 
test
````

Iz sbt Shell-a mozemo izaci komandom:
```` terminaloutput
exit
````
